# Cursor IDE Rules

This directory contains rules and prompts for Cursor IDE AI assistant.

## Files

- **AI_ASSISTANT_PROMPT.md** - Comprehensive development guide for RCAP project
  - SpecKit-driven development workflow
  - Technology stack and constraints
  - Code standards and best practices
  - Security requirements
  - Testing requirements

## Usage

Cursor IDE automatically loads files from `.cursor/rules/` directory to provide context-aware assistance.

For other AI assistants (ChatGPT, Claude, etc.), reference this file directly:
```
.cursor/rules/AI_ASSISTANT_PROMPT.md
```
